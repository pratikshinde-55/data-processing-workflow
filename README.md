# Raw to ML Pipeline

An end-to-end data preprocessing pipeline that transforms raw data into machine learning-ready datasets.

## Features

* Data ingestion and loading
* Data cleaning and missing value handling
* Outlier detection and treatment
* Data transformation and feature engineering
* Encoding categorical variables
* Feature scaling and normalization
* Exploratory Data Analysis (EDA)
* Export of processed datasets for machine learning models

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Workflow

Raw Data → Cleaning → Transformation → Feature Engineering → Scaling → ML Ready Data

## Goal

To build an industry-style data preprocessing pipeline that prepares raw datasets for machine learning applications.
